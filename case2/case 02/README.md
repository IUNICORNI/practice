# Кейс-задача №2

Веб-приложение «Счётчик» на HTML, CSS и JavaScript.

## Что реализовано
- кнопки «Плюс» и «Минус»;
- окно вывода результата;
- изменение цвета окна результата в зависимости от значения;
- блокировка кнопок при достижении значений 10 и -10;
- вывод сообщения о достижении экстремального значения.

Для запуска достаточно открыть файл `index.html` в браузере.
