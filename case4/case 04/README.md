# Кейс-задача №4

Веб-калькулятор на HTML, CSS и JavaScript.

## Реализованные возможности
- два поля ввода;
- четыре кнопки математических операций;
- обработка ввода и вывод результата;
- сообщение об ошибке при вводе нечисловых значений;
- дополнительная защита от деления на ноль.

Для запуска откройте `index.html` в браузере.
