# Кейс-задача №3

Простое веб-приложение на Django, которое принимает имя пользователя, сохраняет его в базу данных и показывает персонализированное приветствие.

## Запуск проекта

```bash
python -m venv venv
venv\Scripts\activate    # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

После запуска откройте адрес, который покажет Django-сервер.
